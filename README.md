程式介紹:
遊戲玩法:
1.  在4x4的網格上進行.
2.  遊戲開始時，會產生隨機兩個數字2或4
3.  玩家可以通過上、下、左、右滑動棋盤來移動所有數字
4.  每次滑動後，所有數字塊會向滑動方向移動
5.  直到達成2048或不能移動時，則遊戲結束
